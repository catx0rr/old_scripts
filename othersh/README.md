Other shell scripts I personally use

- autossh.sh  
    * auto scan and try ssh connection to a subnet
- autostart-mac.sh  
    * create a symlink and execute macchanger every reboot
- debupdate.sh  
    * update automation for deb
- macchanger.sh
    * change your mac-address
- shutdownmenu.sh
    * add a shutdown menu in grub (for deb only)
